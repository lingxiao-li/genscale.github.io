GenScale: A Benchmark for Relative Object Scale in Image Generation and Editing

Evaluating whether modern image generators preserve real-world object proportions, and repairing scale errors with a model-agnostic correction agent.

Lingxiao Li Max Whitton Ledell Wu Boqing Gong

Boston University Creatify Lab

Paper arXiv coming soon Code Website

Modern image generators can produce photorealistic outputs while violating familiar real-world size relationships. GenScale evaluates this failure mode across text-only generation, human-product generation, and scale correction.

Abstract

Modern image generation and editing systems can produce photorealistic, prompt-aligned images, but still often render familiar objects at implausible relative sizes. To measure this failure mode, we introduce GenScale, a benchmark and evaluation protocol for real-world relative object scale in image generation and editing. GenScale contains 900 image-level entries and 1,643 pairwise anchor-target scale relations across common-object generation, human-product generation with metric dimensions, and scale correction from failed generations. We further design a human-calibrated ordinal judge for scalable pairwise scale evaluation. We also introduce Rescale, a model-agnostic post-processing agent for localized scale correction without modifying the source generator. Experiments reveal that state-of-the-art image generators and editors cannot reliably observe relative scale yet, while Rescale consistently improves scale plausibility across generated and edited images.

Benchmark

GenScale represents relative scale as a pairwise anchor-target relation grounded in physical-size metadata. It tests implicit common-object size priors, metric product scale with human anchors, and post-generation scale correction.

Task 1

Common-Object Scale

Text-only prompts test whether models infer plausible relative sizes for familiar objects under natural-depth and same-plane layouts.

Task 2

Human-Product Scale

Reference-conditioned prompts specify product dimensions and human anchors, requiring metric scale realization while preserving product identity.

Task 3

Scale Correction

Erroneous generations become editing tasks that test either autonomous scale-error discovery or exact numeric resize following.

Rescale

Rescale turns the benchmark formulation into an actionable correction interface. Given an image, object identities, and physical-size references, the agent diagnoses scale anomalies, chooses the editable object, predicts a resize factor, and performs localized correction while preserving identity, layout, lighting, and background.

The correction pipeline extracts the target object, completes the background, adjusts masks and depth, reinserts the resized object, and verifies whether another correction round is needed.

Results

GenScale exposes three recurring failure modes: mean regression in common-object generation, product magnification in human-product generation, and weak autonomous diagnosis in scale correction. Rescale reduces scale error across all tested source settings while keeping visual and identity metrics high.
